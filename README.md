# Trailblazer Analytics Marketing Site

Static Astro + React site ready for simple shared hosting (HostGator).  
Generated via ChatGPT **Vibe‑Code Dev Kit**.

## Local dev
```bash
pnpm install
pnpm run dev   # http://localhost:4321
```

## Build & Zip
```bash
pnpm run package   # outputs trailblazer-analytics-dist.zip
```

Upload contents of the zip to your HostGator `public_html/` directory.
